<p>此插件为依赖jquery的一个全屏或者固定宽度的图片轮播特效</p>

<p>其中或有些技术不到位，做得不好的，欢迎指教!</p>

<p>轮播区域为li结构，其中有2块内容</p>

<ul>
	<li>一块背景，命名为slide-bg</li>
	<li>一块文字，，命名为slide-txt</li>
</ul>

<p>可单独对着2块内容做控制当然也可以修改源码，欢迎折腾。</p>
